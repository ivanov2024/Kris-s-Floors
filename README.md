# Kris's Floors

A modern single-page website for **Kris's Floors**, a solo flooring installation business run by Kristian.  
The website is designed to present services clearly, build trust, showcase completed work, and guide visitors to contact through social media.

## Features

- Single-page layout
- Smooth in-page navigation
- Responsive design for mobile, tablet, and desktop
- Services section
- Gallery section
- Reviews section
- Social-media-based contact section
- Clean and professional design focused on a local trade business

## Purpose

This project was created to provide a strong online presence for a flooring installer who specializes in laminate and other flooring types.  
The main goal is to make the business look trustworthy, skilled, and easy to contact.

## Tech / Build

This project was generated and refined using **Stitch AI with Gemini 3 Pro**.

## Project Structure

The website includes the following sections:

- Hero
- About
- Services
- Why Choose Kris’s Floors
- Gallery
- Process
- Reviews
- Contact
- Footer

## Design Goals

- Premium but approachable look
- Consistent spacing and alignment
- Single-page user experience
- Clear navigation between sections
- No unnecessary clutter or multi-page structure

## Contact

The website is designed to direct visitors to contact through:

## Notes

This project focuses on clarity, trust, and presentation for a one-person flooring business.
